What are the different ways to select an element in the DOM?

The different ways of select an element is by using document.getElementById(), document.getElementsByTagName(), document.getElementsByClassName(), document.querySelector(), document.querySelectorAll()

What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns all elements including texts, spacing and html element inside a parent element. innerText returns only texts that are not hidden using css while textContent return all texts even if they are hidden using css.

What is event delegation in the DOM?

Event delegation is a technique of adding the event listener to the parent element instead of adding same event to the multiple children elements.

What is event bubbling in the DOM?

Event bubbling is a process where any sort event happens on the child elements, it bubbles up to the parent and all the way to the root element.

How do you create, add, and remove elements using JavaScript?

document.createElement() creates an element, document.innerHtml() is used to add an element and element.remove() is used to remove an element.
